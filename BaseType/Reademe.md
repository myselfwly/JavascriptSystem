# Javascript基础类型
在JavaScript中有6种基础类型。
1. String —— 字符串
2. Null —— 空值
3. Undefined —— 未定义
4. Number —— 数字
5. Symbol —— 符号
6. Boolean —— 布尔

> javascript 中所有复杂类型均由以上6种基本数据类型构成

## 在 typeof 关键字中的表现
|原始类型|表现结果|
|---|---|
|String|"string"|
|Null|"object"|
|Undefined|"undefined"|
|Number|"number"|
|Symbol|"symbol"|
|Boolean|"boolean"|
